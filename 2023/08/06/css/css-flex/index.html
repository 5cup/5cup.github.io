flex 布局

flex 布局,即”弹性盒布局”, 旨在提供一种更有效的方式来布局、对齐和分配容器中项目之间的空间，即使他们的大小未知或动态变化。

声明定义

通过将 display 属性设置为 flex 或 inline-flex，将父元素声明为 弹性 容器。

flex 容器属性

  • flex-direction
  • flex-wrap
  • flex-flow
  • justify-content
  • align-items
  • align-content
