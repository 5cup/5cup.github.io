JavaScript 数据类型检测方法

  • typeof
  • instanceof
  • construct
  • Object.prototype.toString.call()

typeof

其中数组、对象、null 都会被判断为 object，其他判断正确

instanceof

instanceof 可以正确的判断对象的类型，其内部运行机制是判断在其原型链中能否能找到该类型的原型。

instanceof 只能正确判断引用数据类型，而不能判断基本数据类型。

constructor

constructor 有两个作用，一是判断数据的类型，二是对象实例通过 constructor 对象访问它的构造函数。需要注意的是，如果一个对象的原型被手动修改了，那么就不能用 constructor 来判断数据类型了。
