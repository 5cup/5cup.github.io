    在现有存储库中运行 git init 是安全的。它不会 覆盖已经存在的内容。

分支和合并

  • 列出分支
  • 创建分支
  • 删除分支
  • 切换分支
  • git branch [--list] - 列出现有分支(本地)
    当前分支将以绿色突出显示，并用*标记。

    git branch
  • git branch -r - 列出远程跟踪分支

    git branch -r
  • git branch -a - 同时显示本地和远程分支

    git branch -a
  • git branch <branchName> - 创建新分支

    git branch <branchName>
    1. 输入git branch命令可以看到目前只有一个 main 分支
    2. 输入git branch branch1 创建新分支 branch1
    3. 再次查看能看到刚创建的新分支了
  • git checkout -b <branchName> - 创建新分支
  • git branch -B <branchName> - 创建新分支
  • git branch -d <branchName> - 删除指定分支（本地）

    git branch -d <branchName>
    1. 输入git branch可以看到目前拥有两个分支：main、branch1
    2. 输入git branch -d branch1删除 branch1 分支
    3. 再次查看只有 main 分支了
  • git branch -d -r <branchName> - 删除指定分支（远程）

    git branch -d -r <branchName>
    1. 输入git branch -a可以看到远程仓库拥有两个分支：main、branch1
    2. 输入git branch -d -r origin/branch1删除 branch1 分支
    3. 再次查看只有 main 分支了
  • git checkout <branchName> - 切换到(已存在的)指定分支
  • git checkout -b <branchName> - 切换到指定分支（若该分支不存在则新建该分支）
