故障风格文字效果

发表于 2024-06-26 更新于 2024-08-21 本文字数： 889 阅读时长 ≈ 3 分钟

原理

利用 clip-path 属性对叠加在一起的相同文字进行随机裁切，加上动画就能形成这种效果了。

示例

  1. html 部分就一个 div
    定义了一个data-text属性且值与要显示的文字相同 HELLO WORLD

  2. 先写一些基本样式设置字体大小和居中显示等

  3. 添加两个与要显示的内容一样的伪元素，在与要显示的内容叠加的同时并在水平方向形成偏移

    右边部分会有突出的阴影，等添加完动画就不明显了，可以暂时忽略。

  4. 利用 clip-path 属性创建动画
